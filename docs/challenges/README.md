# Challenge Strategy

This directory contains the product strategy for turning Question Chains into short, replayable exam-marking games.

The current recommendation is:

- make Biology + Physics the complete initial wedge;
- lead publicly with Biology;
- use Physics as the highest-trust scoring control;
- expand the same Science shell into Chemistry;
- consider Maths later through a separate working-focused interface;
- treat Combined Science as a qualification filter rather than a subject taxonomy.

The detailed reasoning, UK demand evidence, subject comparison, game formats, user experience, acquisition model, and validation plan are in:

- [Subject Priority and Challenge Strategy](./subject-priority-and-challenge-strategy.md)
