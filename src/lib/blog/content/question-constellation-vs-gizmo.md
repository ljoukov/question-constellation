## Quick comparison

| Feature                 | Question Constellation                                               | Gizmo                                                      |
| ----------------------- | -------------------------------------------------------------------- | ---------------------------------------------------------- |
| Main use                | Exam-question transfer through answer chains                         | AI flashcards and spaced memory practice                   |
| Best content type       | Mark-scored written responses and linked exam problems               | Definitions, facts, formulae and recall prompts            |
| Revision problem solved | Knowing what a mark-scoring answer needs to say                      | Remembering material reliably over time                    |
| Exam board detail       | Pages carry board, paper, topic, tier and mark value where available | Flashcard sets depend on the content selected or generated |
| Best pairing            | Use after recall practice to test exam application                   | Use before exam questions to strengthen memory             |

## Memory still matters

AI flashcards can help students keep facts alive. That is useful for definitions, formulae, vocabulary and small recall questions.

Written GCSE answers need the next step. The student has to choose the right fact, link it to the question, and explain the consequence.

## The answer-chain layer

Question Constellation makes that link structure visible. The student sees the answer chain, then practises related questions that use the same chain in another context.

That is different from remembering a card. It is practice in applying what the student remembered.

## Best way to use both

Use Gizmo for recall sessions. Use Question Constellation when the student is ready to attempt exam questions and compare the answer with a mark checklist.

## Verdict

Gizmo supports memory. Question Constellation supports exam application. A good revision session can include both.
