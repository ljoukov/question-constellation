## Quick comparison

| Feature            | Question Constellation                              | Tassomai                                                     |
| ------------------ | --------------------------------------------------- | ------------------------------------------------------------ |
| Main use           | Written exam-answer practice around reusable chains | Daily quiz questions designed to build recall and confidence |
| Best question type | Applied, multi-step and written-response questions  | Short recall checks and knowledge questions                  |
| Feedback focus     | What reasoning link the answer missed               | Whether the learner selected or recalled the expected answer |
| Student habit      | Deliberate exam practice sessions                   | Frequent low-friction quiz routines                          |
| Best fit           | Students who need to improve exam answers           | Students who need regular knowledge reinforcement            |

## Recall and written answers

Regular recall practice can be valuable. A student who cannot recall core facts has little to build with in an exam.

The GCSE problem is that many marks sit beyond recall. The student may need to connect a fact to a process, then to evidence, then to the effect in the question.

## What Question Constellation adds

Question Constellation shows the chain that connects those links. It also connects questions that use the same chain, so the student can practise transfer rather than repeat one answer.

That makes it a useful follow-up to quiz-led revision.

## Best way to use both

Use Tassomai or another recall routine to keep facts active. Use Question Constellation when the next step is building a mark-scoring answer from those facts.

## Verdict

Tassomai supports recall habits. Question Constellation supports answer construction. GCSE students often need both memory and structure.
