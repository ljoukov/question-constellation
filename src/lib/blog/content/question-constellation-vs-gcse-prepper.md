## Quick comparison

| Feature        | Question Constellation                                               | GCSE Prepper                                               |
| -------------- | -------------------------------------------------------------------- | ---------------------------------------------------------- |
| Main use       | Public question chains, mark checklists and transfer practice        | AI-assisted GCSE practice and marking tools                |
| Core object    | The answer chain shared by related questions                         | The marked answer or practice task                         |
| Best for       | Students who want to understand the pattern behind several questions | Students who want quick feedback on a specific answer      |
| Feedback style | Checklist plus missing-link language tied to exam logic              | AI feedback depends on the submitted task and product flow |
| Best pairing   | Use to build a reusable exam-answer habit                            | Use to check an answer when a marking flow is available    |

## Why reusable chains matter

Feedback on one answer can be helpful. The harder GCSE problem is using that feedback next time, when the wording changes.

Question Constellation solves that by making the chain the main object. The student can move from one question to another and see the same logic under a new surface.

## Where AI marking can help

AI-assisted marking can be useful when it points students back to specific criteria. The risk is vague feedback that sounds helpful but does not change the next answer.

Question Constellation keeps the mark checklist and chain close to the question, so feedback has a concrete structure.

## Best way to use both

Use an AI marking flow to check a draft. Use Question Constellation to understand which chain the answer needs and practise another question that uses it.

## Verdict

GCSE Prepper is closer to AI-assisted marking. Question Constellation is closer to a public answer-chain atlas. The useful question is whether the student needs feedback on one answer or a reusable pattern for the next one.
